This script takes the blast2go, interpro, and annotation input and merges them together into one file, please place the three files into the "input" folder before starting

***NOTE***

The files in the input directory must be named in the following manner:

annotation.tsv ---> the annotation file
blast2go.txt   ---> the blast2go file
interpro.raw   ---> the interpro file

***NOTE***


When the script finishes running, the output file will be availible in the output directory.

(output file name is combined_annotation.tsv)
